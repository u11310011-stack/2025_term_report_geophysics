第九週：板塊構造理論 (Plate Tectonics)

地球表層的動態系統：圈層結構與邊界類型

岩石圈 vs. 軟流圈

岩石圈 (Lithosphere)： 由地殼與上地函最頂部組成，表現為堅硬且易碎的固體。它被分割成數個巨大的「板塊」。

軟流圈 (Asthenosphere)： 位於岩石圈下方，處於一種「塑性」或「半融熔」狀態。雖然它是固體，但能像極慢的液體般對流。

板塊驅動力

  • 地函對流 (Mantle Convection)
  • 中洋脊推力 (Ridge Push)
  • 板塊拉力 (Slab Pull) —— 最主要的動力來源

三大板塊邊界類型

Divergent

張裂型邊界

板塊彼此分離，岩漿自地函湧出形成新地殼。

  • 例：大西洋中洋脊
  • 例：東非大裂谷
Convergent

聚合型邊界

板塊碰撞或隱沒，伴隨劇烈地震與造山運動。

  • 例：喜馬拉雅山 (陸-陸)
  • 例：馬里亞納海溝 (海-海)
Transform

錯動型邊界

板塊水平滑動，既不產生也不破壞地殼。

  • 例：聖安德烈斯斷層
  • 通常連接中洋脊段落

第十週：Pygmt 地圖繪製實務

掌握地球科學最強大的繪圖工具：Python interface for Generic Mapping Tools

什麼是 PyGMT?

PyGMT 是 GMT (Generic Mapping Tools) 的 Python 介面，它能產生高品質、出版等級的地理圖表。在地球物理學中，常用於繪製震央分佈、地形圖及板塊邊界。

核心功能與地圖投影

  • 投影法： 麥卡托投影 (M)、圓柱投影 (X)、極投影
  • 海岸線： 使用 fig.coast() 快速設定陸地與海洋顏色
  • 數值高程： 內建支援地形陰影與高程資料顯示
查看完整線上講義
# 範例程式：快速繪製台灣地圖
import pygmt

fig = pygmt.Figure()
# 設定區域 (Region) 與 投影 (Projection)
fig.coast(
    region=[119, 123, 21, 26],
    projection="M10c",
    land="lightgray",
    water="skyblue",
    shorelines=True,
    frame="ag"
)
fig.show()
重點提示：region 參數為 [西, 東, 南, 北] 經緯度範圍。

第十一週：地磁學專題 —— 磁極飄移與地磁逆轉

當前地磁活動概況

地磁極並非固定不動，而是隨著外地核液態金屬的流動而不斷遷移。近年來，科學家觀察到磁北極正以異常的速度離開加拿大向俄羅斯西伯利亞方向移動。這種「磁極飄移」現象是研究地球內部動力的關鍵指標。

55 km/y 當前磁北極移動速度
780,000 yr 距離上次大逆轉時間
-10% / 百年 地磁強度衰減率

磁場的產生機制

地球磁場主要由外地核的液態鐵鎳流動所產生的「發電機效應」維持。這種流動受到地球自轉與熱對流的影響，形成複雜的三維磁場結構。當對流模式發生劇烈改變時，便可能引發磁極飄移加速甚至磁極逆轉。

磁極逆轉的文明影響

  • • 導航精度失效： 航空與航海導航必須頻繁修正偏角。
  • • 通訊設施損毀： 磁場減弱會導致低軌道衛星暴露在高能粒子下。
  • • 能源架構脆弱： 誘發地磁感應電流會對長距離輸電網路造成巨大壓力。

關鍵術語解析

發電機效應 (Dynamo Effect)

液態外地核中導電流體的運動轉化為磁能的物理過程。

南大西洋異常區 (SAA)

地磁場特別虛弱的區域，是研究磁場逆轉前兆的重要觀測點。

世界地磁模型 (WMM)

每五年更新一次的全球標準導航模型，近年因磁極移速過快曾提前更新。

本週課後任務

請查閱 NOAA 地磁觀測數據，分析過去 50 年磁北極移動軌跡的加速度變化，並撰寫一份關於其對全球定位系統（GPS）影響的簡報。

第十二週：地球的熱 (Earth's Heat)

板塊構造與熱分布探討

地熱流量：1.4 × 10²¹ J

熱從何而來？

地球內部的熱主要來自放射性元素衰變與原始積聚熱，透過傳導、對流與輻射降溫。

內部熱的影響？

驅動板塊構造、地幔對流，並產生地球磁場（發電機效應）。

外部熱的影響？

太陽輻射加熱地表，驅動大氣循環、風化作用與氣候系統。

年度能量收支 (Energy Budget)

成分 能量 (J) 比例比
太陽輻射與反射 5.4 × 10²⁴ ≈ 4000
地球內部地熱流 1.4 × 10²¹ 1
潮汐摩擦 ≈ 10²⁰ ≈ 0.1
地震彈性能 ≈ 10¹⁹ ≈ 0.01

結論：地內熱流量僅佔太陽輻射的 1/4000。

熱力學基礎 Thermodynamics

第一定律：能量守恆

ΔQ = TΔS = ΔU + ΔW

系統熱能增加 = 內能增加 + 對外做功 (如體積膨脹)

重要定義
  • 焦耳 (J)： 能量單位。
  • 卡路里 (cal)： 1g水升溫1°C所需熱量 (≈ 4.1868 J)。
  • 比熱 (cₚ)： ΔQ = cₚ m ΔT
  • 體積膨脹係數 (α)： 1/V (∂V/∂T)ₚ

地球內部溫度梯度

  • 地表梯度：約 30 K/km
  • 地核溫度：約 6000 K
地球溫度剖面示意 (Crust → Mantle → Core)

岩石/構造類型 生熱率 (nW/kg)

地表熱流分析 (Earth Heat Flow)

熱流來源比例 (%)

熱源來源 大陸 海洋
板塊冷卻 (Cooling) 20% 85%
地函底部熱流 25% 10%

第十三週：深層地熱探勘系統

從探勘到驗證：以台東延平案場為例

綜合性調查 (地質、地化、地物)

地質 (Geology)

尋找熱的「通道」。透過線型分析與斷裂系統調查，鎖定主要的東西向或北東向裂隙。

地球化學 (Geochemistry)

分析熱的「來源」。分析水的起源、溫度與循環路徑，判斷熱水循環系統。

地球物理 (Geophysics)

描繪熱的「形貌」。利用電磁法 (MT/AMT) 描繪儲集層的空間分布與幾何形狀。

典型地熱系統模式

  • 1
    補給區 (Recharge Zone)： 地表水滲透至地底深處，由高地溫梯度加熱。
  • 2
    上湧區 (Up-flow Zone)： 系統的核心，熱水沿裂隙快速上升。此為產能井的最佳位置 (如儲集層 UF02)。
  • 3
    外滲區 (Out-flow Zone)： 熱水橫向流動並與冷水混合，通常為溫泉露頭所在地。

探勘實例：延平案場 UF02 儲集層
