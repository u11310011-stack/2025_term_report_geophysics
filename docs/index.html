熊本地震 (2016)

規模 M7.3 主震。黃博士實地經歷震度 7 的衝擊，觀察到複合式災害（山崩、斷層破裂、火山噴發預警）。

阿蘇火山監測

利用地震儀、次聲波(Infrasound)與地殼變形監測，預警斯通波利式與岩漿水蒸氣噴發。

嘉南地區觀測

建立西南台灣即時地震觀測網 (NC, IES-BATS)，針對布田川-日奈久斷層帶與嘉南平原地震序列進行分析。

大屯火山群省思

台灣大台北都會區緊鄰大屯火山與山腳斷層，應加強地表噴氣增加與水蒸氣噴發之日常監測。
