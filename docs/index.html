第十週：PyGMT 與地球物理視覺化

PyGMT 是 GMT 的 Python 介面，廣泛應用於地形、地震、板塊與地球物理資料視覺化， 是本課程期末專題的重要工具。

地形 DEM

使用 grdimage 繪製地形。

地震資料

利用 plot 疊加震央。

圖層流程

Figure() → show() 


import pygmt

fig = pygmt.Figure()
fig.grdimage(
    grid="@earth_relief_15s",
    region=[119, 123, 21.5, 25.5],
    projection="M15c",
    cmap="geo"
)
fig.colorbar(frame=["x+lElevation", "y+lm"])
fig.show()
